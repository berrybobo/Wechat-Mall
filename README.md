# 蔓悦果礼 - 微信小程序

一个结合美甲服务与水果礼盒配送的创新微信小程序，为用户提供一站式的指尖之美与甜蜜体验。

## 项目介绍

"蔓悦果礼"是一款专为美甲门店打造的小程序，将美甲服务与水果礼盒配送相结合。用户在做美甲的同时，可以便捷地购买搭配好的水果礼盒，作为给自己或亲友的甜蜜礼物。

## 功能特性

### 1. 首页展示
- 店铺信息展示（名称、标语、特色服务）
- 送礼流程介绍
- 产品分类浏览（热门、推荐）
- 产品列表展示（带分类筛选）
- 购物车入口与数量提示

### 2. 产品详情
- 产品详细信息展示（标题、价格、描述、规格）
- 产品标签与特色介绍
- 产品搭配建议（美甲色系、水果组合）
- 服务保障说明

### 3. 购物车
- 购物车商品管理（添加、删除、修改数量）
- 购物车商品总价计算
- 购物车数量与总价展示
- 清空购物车功能

### 4. 订单流程
- 订单确认（地址、商品、金额）
- 订单提交
- 订单成功提示

### 5. 个人中心
- 用户信息展示
- 订单管理入口

## 使用说明

1. **首页浏览**：进入小程序后，您可以浏览店铺信息、送礼流程和产品列表。
2. **产品筛选**：使用顶部标签（热门/推荐）和分类筛选产品。
3. **查看详情**：点击产品卡片查看产品详细信息。
4. **加入购物车**：在产品列表或详情页点击加入购物车按钮。
5. **购物车管理**：点击底部TabBar的购物车图标，管理购物车中的商品。
6. **提交订单**：确认购物车商品后，点击结算按钮提交订单。
